# Test File Generator

## Overview
The Test File Generator is a static website that allows users to select and generate different testing files. Users can choose from various file formats and specify the desired file size. The generated files are intended for testing purposes and contain random data.

## Project Structure
```
test-file-generator
├── css
│   └── styles.css        # Contains styles for the website
├── js
│   └── script.js         # Contains JavaScript for handling user interactions
├── index.html            # Main HTML document for the static website
└── README.md             # Documentation for the project
```

## Features
- Select file format: JPG, PNG, GIF, PDF, DOCX
- Specify file size with units: KB, MB, GB
- Generate button to create the selected file

## Installation
1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd test-file-generator
   ```
3. Open `index.html` in a web browser to view the application.

## Usage
1. Choose the desired file format from the dropdown menu.
2. Enter the file size in the input field and select the size unit.
3. Click the "Generate" button to create the testing file.
